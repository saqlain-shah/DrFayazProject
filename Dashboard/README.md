## Description

The Dental Admin Dashboard Template is a comprehensive and feature-rich solution designed specifically for clinic. It provides a powerful and intuitive interface for managing various aspects of clinic operations, streamlining administrative tasks, and enhancing overall efficiency.

## Installation

1. Open folder in code editor

2. Install project dependencies:

```shell
npm install
```

## Usage

1. Start the development server:

```shell
npm start
```

2. Open your browser and visit `http://localhost:3000` to view the application.
